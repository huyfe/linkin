Tổng hợp các API cần xây dựng:
1. User (login, sign up, log out, reset password, role) -> Lam
2. Link (create, read, update, delete, share to another, set private/public, report) -> Huy
3. Category (create, read, update, delete, share to another, set private/public) -> Lập
4. Group (create, read, update, delete, push into a link, create category, join, out, report) -> Hải
5. Timeline (for group, for user) -> Đạt
